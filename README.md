# U-Net

A U-Net implementation for image segmentation on the [Oxford-IIIT Pet Dataset](https://www.robots.ox.ac.uk/~vgg/data/pets/).

## What is U-Net?

U-Net is a convolutional neural network designed for image segmentation. It uses an encoder-decoder structure with skip connections to produce precise pixel-wise predictions.

## Usage

Open and run the notebook:

```bash
[unet.ipynb](https://github.com/sudikshyapant/unet/blob/main/unet.ipynb)
```

## Dataset

This project uses the Oxford-IIIT Pet Dataset, which contains images of 37 pet breeds with pixel-level segmentation masks.
